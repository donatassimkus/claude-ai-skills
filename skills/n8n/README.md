# n8n for Claude Code

**Knowledge skill.** The complete method for building reliable n8n automations: deterministic workflow design, error handling, AI-agent workflows, common recipes, and a debugging playbook.

## Install

Copy [`INSTALL-PROMPT.md`](INSTALL-PROMPT.md) and paste it into your AI agent (Claude Code, Cursor, or similar). It installs [`SKILL.md`](SKILL.md) unchanged, asks one question (n8n Cloud or self-hosted), then designs your next workflow or debugs a broken one. No accounts, no keys.

## What's here

- `INSTALL-PROMPT.md`: the runbook you paste.
- `SKILL.md`: the full method your AI installs.
