# Chrome Extension for Claude Code

**Knowledge skill.** The complete method for shipping a Chrome extension through Web Store review with the least friction: the popup-only Manifest V3 architecture, icon generation, the popup pattern, every Chrome Web Store form field with the answer to put in it, and the seven reasons extensions get rejected.

## Install

Copy [`INSTALL-PROMPT.md`](INSTALL-PROMPT.md) and paste it into your AI agent (Claude Code, Cursor, or similar). It installs [`SKILL.md`](SKILL.md) unchanged, asks one question about what you are shipping extensions for (a brand asset, a product for users, or an internal tool), then builds one of your own end to end or diagnoses a rejection. No accounts, no keys.

## What's here

- `INSTALL-PROMPT.md`: the runbook you paste.
- `SKILL.md`: the full method your AI installs.
