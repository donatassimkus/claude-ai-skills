# SEO Hacks for Claude Code

**Knowledge skill.** A routed library of 152 white-hat tactical SEO wins, each with execution steps and its original source, across traditional search, AI search, LLM answers, community, platforms, brand and PR, and technical SEO.

## Install

Copy [`INSTALL-PROMPT.md`](INSTALL-PROMPT.md) and paste it into your AI agent (Claude Code, Cursor, or similar). It installs the library unchanged, asks one question about your business type, then returns the top five highest-impact tactics for your site. No accounts, no keys.

## What's here

- `INSTALL-PROMPT.md`: the runbook you paste.
- `SKILL.md`: the router your AI installs. It indexes every hack and loads only the reference a task needs.
- `references/`: the tactics grouped by channel, loaded on demand.
