### 129. Audit AI crawler readability [source](https://www.instagram.com/p/DK2iv_iCLhA/) · Jun 2025

`technical-seo`, `ai-seo`, `content-strategy`, `llm-seo`

**What it does:** Test whether AI systems can accurately understand your pages by feeding raw page text into an LLM to check comprehension.

**How to execute:**
1. Select 5-10 key pages on your site (homepage, top landing pages, pillar content)
2. Copy the full raw text from each page including navigation, sidebar, and footer
3. Paste into ChatGPT or Claude and ask: 'What is this page about and what are the key points?'
4. Compare the AI summary to your intended message and value proposition
5. If the AI misunderstands any page, clean up that page's content structure and re-test

**Why it works:** AI companies are crawling sites for training and citation, so confirming your content reads correctly to LLMs ensures accurate representation in AI answers.

### 144. Inspect ChatGPT Search Queries [source](https://www.instagram.com/p/DO9N-GCDEV_/) · Sep 2025

`ai-seo`, `keyword-research`, `tools-workflows`, `llm-seo`

**What it does:** Reverse-engineer the exact search queries ChatGPT uses when browsing the web to answer a prompt, so you can optimize content for AI-driven search.

**How to execute:**
1. Ask ChatGPT a question that triggers a web search
2. Copy the string from the URL after /c/ (the conversation ID)
3. In Chrome, right-click the page and select Inspect
4. Go to the Network tab at the top of DevTools
5. Refresh the page to capture network requests
6. In the Network filter bar, paste the conversation ID string you copied
7. Click the matching network request
8. Go to the Response tab
9. Search the response code for the word 'queries'
10. Read the actual keyword queries ChatGPT used to search on your behalf

**Why it works:** Knowing the exact queries ChatGPT generates lets you optimize content for AI-recommended results, a new SEO surface most competitors ignore.

### 163. Reverse-engineer ChatGPT search queries [source](https://www.instagram.com/p/DSQnlsPDChs/) · Dec 2025

`ai-seo`, `content-strategy`, `keyword-research`, `llm-seo`

**What it does:** Use Chrome DevTools to extract the exact queries ChatGPT submits to the web when answering prompts, then use that language in your content.

**How to execute:**
1. Go to ChatGPT and enter a prompt relevant to your niche (e.g. 'fashion trends for this winter')
2. Right-click the page and select Inspect to open Chrome DevTools
3. Click the Network tab
4. Copy the string that appears after /c in the page URL
5. Paste that string into the Network tab filter box
6. Refresh the page
7. Click the entry with orange brackets matching the code you filtered
8. Go to the Response tab
9. Search (Ctrl+F) for the word 'queries'
10. Note the exact search strings ChatGPT used to retrieve web results
11. Use those exact phrases as H2 headings in existing content, or create new content targeting those queries directly

**Why it works:** ChatGPT surfaces brands whose content matches the literal queries it sends to the web, so mirroring that language increases the chance your content gets cited.

### 164. Mine Perplexity queries for H2s [source](https://www.instagram.com/p/DSXXSfdjBoQ/) · Dec 2025

`ai-seo`, `on-page-seo`, `content-strategy`, `llm-seo`

**What it does:** Use Perplexity's 'Steps completed' panel to extract the exact search queries AI runs when answering a prompt, then use that language as H2 headings or new content pieces.

**How to execute:**
1. Go to Perplexity.ai and search for a query relevant to your target topic
2. After the answer loads, click 'Steps completed' (shown during the answer generation)
3. Copy every search query Perplexity ran internally to build its answer
4. Use those exact phrases as H2 headings within your existing content
5. Or create entirely new content pieces targeting each query as its own article
6. Repeat for multiple prompts across your topic cluster

**Why it works:** AI answers are built from specific sub-queries — matching your content structure to those queries signals relevance to the AI retrieval layer, increasing the chance your page gets cited.

### 174. Filter AI-assisted searches in GSC [source](https://www.instagram.com/p/DTYpj_-jPB_/) · Jan 2026

`google-seo`, `keyword-research`, `analytics`, `llm-seo`

**What it does:** Use Google Search Console query filters to isolate long-tail (7+ word) queries that come from LLM-assisted searches.

**How to execute:**
1. Open Google Search Console
2. Go to Performance > Search results
3. Click 'Add Filter' then 'Query'
4. Set filter to 'Contains' and enter a regex or phrase that captures 7+ word queries (e.g. filter by minimum word count using regex '(\w+ ){6,}')
5. Review the resulting list — these are AI-assisted search queries

**Why it works:** LLMs generate longer, more conversational queries, so filtering for 7+ word searches surfaces keywords you already rank for that have AI-search intent.
