# Ingest for Claude Code

**Agent skill.** Drop any source (PDF, video, podcast, article) and your AI reads, distils, and files it.

## Install

Copy [`INSTALL-PROMPT.md`](INSTALL-PROMPT.md) and paste it into your AI agent (Claude Code, Cursor, or similar). It detects what you already have, builds the skill on your own machine, and runs it.

## What's here

- `INSTALL-PROMPT.md`: the runbook you paste.
