## Install

Copy [`INSTALL-PROMPT.md`](INSTALL-PROMPT.md) and paste it into your AI agent (Claude Code, Cursor, or similar). It installs the skill files unchanged, asks one question about your business type, then runs the method on a real page of yours. No accounts, no keys.
